## 📖 Overview

This repository tracks notable **SaaS platforms** and **open-source projects** building the next generation of **Email Deliverability AI Toolkits**. These tools leverage artificial intelligence and machine learning to:
- 🛡️ Monitor domain and IP reputation in real-time.
- 🕵️ Analyze spam scores and provide automated content optimization.
- ⚙️ Manage complex protocols like **DMARC, DKIM, and SPF**.
- 🚫 Detect blacklisting and trigger automated remediation.
- 📈 Optimize sending infrastructure for maximum inbox placement.
- 📉 Reduce bounce rates and improve sender scores.

**Ideal for**: Email Marketers, Growth Hackers, Deliverability Specialists, and E-commerce Teams.

### 🔍 Specialized Tools

- **[Google Postmaster Tools](https://postmaster.google.com/)**  
  Essential for monitoring your domain's reputation with Gmail, providing data on spam rate, encryption, and delivery errors. 📧

- **[Microsoft SNDS (Smart Network Data Services)](https://sendersupport.olc.protection.outlook.com/snds/)**  
  Provides insights into how your sending IP is viewed by Outlook.com, including spam complaint rates and "trap" hits. 💻

- **[Validity (Everest)](https://www.validity.com/everest/)**  
  Enterprise-grade deliverability platform that combines seed list testing, reputation monitoring, and list verification. 🏔️

- **[SenderScore](https://www.senderscore.org/)**  
  A free service by Validity that scores your IP reputation on a scale of 0-100, similar to a credit score for email. 💯

## 🛠️ Open-Source GitHub Projects

### 🛡️ Reputation & Deliverability Tools

- **[OpenDMARC](https://github.com/trusteddomainproject/OpenDMARC)**  
  Open-source DMARC implementation for policy enforcement, reporting, and authentication monitoring. 🔐

- **[SpamAssassin](https://github.com/apache/spamassassin)**  
  The most widely used open-source spam filtering system with powerful rule-based and Bayesian analysis. 🗡️

- **[rspamd](https://github.com/rspamd/rspamd)**  
  Fast, powerful open-source spam filtering system with neural networks, AI-like scoring, and excellent deliverability features. 🧠

- **[Postfix](https://github.com/vmailing/postfix)**  
  Leading open-source mail transfer agent with extensive configuration for high deliverability. 📬

- **[Mailcow](https://github.com/mailcow/mailcow-dockerized)**  
  Complete open-source mail server stack with built-in monitoring, DMARC, and deliverability tools. 🐄

- **[dmarcian / OpenDMARC tools](https://github.com/dmarcian)**  
  Open-source tools and scripts for DMARC analysis and reporting. 📊

- **[Haraka](https://github.com/haraka/Haraka)**  
  High-performance open-source SMTP server with plugin support for advanced deliverability and security. ⚡

### 💡 Advanced OS Options

- **[Exim](https://github.com/Exim/exim)** — Highly configurable MTA with strong deliverability track record.
- **[Dovecot](https://github.com/dovecot/core)** — Secure IMAP/POP3 server often paired with deliverability tools.
- **LangChain + LLM pipelines** for AI-powered email content analysis and spam risk scoring.
- **Custom Python scripts** using `dnspython`, `spamassassin`, and `pyspf` for automated deliverability checks.
